# FinOptima - Автоматизированный финансовый консультант

## Описание проекта
Веб-сервис для учета личных финансов и получения автоматизированных рекомендаций по оптимизации бюджета. Разработан с использованием фреймворка Django и средств вайб-кодинга (Cursor, большие языковые модели).

## Функционал
- 📊 Учет доходов и расходов
- 📈 Анализ финансовой статистики  
- 🤖 Персонализированные рекомендации на основе анализа расходов
- 🔐 Авторизация пользователей
- 📱 Адаптивный веб-интерфейс

## Технологии
- **Backend:** Python 3.14 + Django 5.2.7
- **Frontend:** Django Templates + Bootstrap 5
- **Database:** SQLite
- **Deployment:** Development server

## Установка и запуск

```bash
# Создание виртуального окружения
python -m venv venv
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
